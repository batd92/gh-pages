# Install Ubuntu

## Windows or macOS with Intel
  - **Install**: https://elroydevops.tech/cach-dung-vagrant-tao-may-ao/

## macOS with Apple silicon
  - **Install Multipass**: 
    ```copy
    brew install --cask multipass
    ```

  - **Check version**: 
    ```copy
    multipass version
    ```

  - **Create & Lunch Multipass VM**: 
    ```copy
    multipass launch --cpus 2 --disk 20G --memory 2G --name Ubuntu-S1
    ```

  - **Check running VMs**: 
    ```copy
    multipass list
    ```

  - **Connect to VM shells**: 
    ```copy
    multipass shell Ubuntu-S1
    ```

  - **Update Ubuntu System**: 
    ```copy
    sudo apt update
    ```

  - **Install xubuntu-desktop for GUI (optional)**: 
    ```copy
    sudo apt install xubuntu-desktop
    ```

  - **Restart**: 
    ```copy
    multipass restart Ubuntu-S1
    ```

# Basic Linux Command Guide

## System Information
- **`pwd`**: Current directory path.
- **`whoami`**: Current user's name.
- **`history`**: List of previously executed commands.

## File and Directory Management
- **`cd [path]`**: Change directory.
- **`ls`**: List files/directories.
- **`mkdir [name]`**: Create a directory.
- **`touch [name]`**: Create/update a file.
- **`rm [name]`**: Delete a file.
- **`cp [source] [destination]`**: Copy a file.
- **`mv [source] [destination]`**: Move/rename a file.

## Managing Permissions
- **`chmod [permissions] [file]`**: Change file permissions.
- **`chown [user]:[group] [file]`**: Change file ownership.

## Searching and Filtering
- **`find [path] -name [name]`**: Search for files.
- **`grep [string] [file]`**: Search for a string in a file.

## Process Management
- **`ps`**: Display running processes.
- **`kill [PID]`**: Terminate a process.

## Disk Information
- **`df`**: Show disk space usage.
- **`du [path]`**: Check size of a directory/file.

## Downloading and Transferring Files
- **`wget [URL]`**: Download a file.
- **`curl [URL]`**: Retrieve information from the web.
- **`scp [file] [user@host:/path]`**: Copy files over SSH.

## Compression and Extraction
- **`tar -cvf [archive] [directory]`**: Compress a directory.
- **`tar -xvf [archive]`**: Extract an archive.

## Text Editors
- **`nano [file]`**: Open file in nano.
- **`vim [file]`**: Open file in vim.

## Remote Connection
- **`ssh [user@host]`**: Connect to a remote server.
